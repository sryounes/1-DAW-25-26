# 2. Sintaxis de XML
La sintaxis de XML es muy estricta para garantizar que los datos sean procesables de forma fiable:
- **Debe tener un elemento raíz**: Todo documento XML debe contener un elemento que sea "padre" de todos los demás.
- **Todas las etiquetas deben cerrarse**: Si abres `<tag>`, debes cerrarla con `</tag>`.
- **Diferencia entre mayúsculas y minúsculas**: La etiqueta `<Carta>` es diferente a `<carta>`.
- **Anidación correcta**: Los elementos deben cerrarse en el orden inverso al que se abrieron.
- **Atributos entre comillas**: Los valores de los atributos siempre deben ir entre comillas (simples o dobles).
