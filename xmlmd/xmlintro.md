# Introducción a XML

XML (eXtensible Markup Language) es un lenguaje de marcado diseñado para almacenar y transportar datos. A diferencia de HTML, XML no hace nada por sí mismo; solo estructura la información.

## 1. Introducción a XML
XML es un lenguaje de marcado similar a HTML, pero con diferencias clave:
- **XML fue diseñado para transportar datos**, no para mostrarlos.
- **Las etiquetas no están predefinidas**: Tú debes definir tus propias etiquetas (ej. `<nota>`, `<mensaje>`).
- **Es autodescriptivo**: El lenguaje está diseñado para que tanto humanos como máquinas puedan leerlo fácilmente.
- **Es un estándar de la W3C**: Es una recomendación oficial del World Wide Web Consortium.
