# 3. El Árbol XML (XML Tree)
Los documentos XML forman una estructura de árbol que comienza en la **raíz** y se ramifica hasta las **hojas**.
- **Raíz (Root)**: El elemento superior del documento.
- **Hijos (Children)**: Elementos contenidos dentro de otro.
- **Padres (Parents)**: Elementos que contienen otros elementos.
- **Hermanos (Siblings)**: Elementos que están en el mismo nivel bajo el mismo padre.

Ejemplo de estructura:
```xml
<libreria>             <libro>              <titulo>XML</titulo> <autor>W3S</autor>   </libro>
</libreria>
